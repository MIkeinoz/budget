One-time setup so your budget
syncs between phones for free.

Step 1 — Create Firebase Project

On a computer, go to console.firebase.google.com
→ Click Add Project → Name it (e.g. "our-budget") → Click through to create.

Step 2 — Enable Realtime Database

In your project → Build → Realtime Database → Create Database
→ Pick any location → Start in Test mode → Click Enable.

Step 3 — Enable Anonymous Auth

Go to Build → Authentication → Sign-in method
→ Enable Anonymous → Save.

Step 4 — Get your config

Go to Project Settings (gear icon) → Scroll to Your apps
→ Click the </> Web icon → Register app → Copy the firebaseConfig object.

Paste the whole object including the curly braces. Remove const firebaseConfig = and the trailing semicolon if present.
